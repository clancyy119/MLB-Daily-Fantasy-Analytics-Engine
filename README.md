# MLB-Daily-Fantasy-Analytics-Engine
A predictive analytics engine for MLB Daily Fantasy Sports that aggregates real-time player data via REST APIs to model optimal lineup matchups.

The Problem: DFS players struggle to aggregate real-time pitcher/batter splits across the league to find high-value matchups.

The Solution: A dashboard that automates data collection and applies a custom scoring algorithm to highlight optimal daily lineups.

Key Features:

REST API Integration: Fetches real-time MLB statistics.

Predictive Logic: Weights historical performance against current game-day variables.

Data Cleaning: Normalizes varying player names and team IDs across data sources.

Tech Stack: Python, Base44 Backend, REST APIs.
